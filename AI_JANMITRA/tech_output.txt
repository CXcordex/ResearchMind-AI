1. System Architecture Overview
AI JanMitra is built as a three-tier architecture: a Node.js/React client layer, a Python FastAPI orchestration and AI layer, and a Supabase PostgreSQL data layer. All AI reasoning is performed by models accessed via OpenRouter and HuggingFace APIs. Every component uses free-tier or open-source resources.

8.5 Step 4 — Live Testing Checklist
Text query test: Type 'What is the BNS?' — verify response is in plain language with legal disclaimer
Voice test: Click microphone, speak in Hindi — verify transcription and Hindi response
Document upload test: Upload a sample PDF notice — verify OCR extraction and scam analysis
Scheme test: Enter profile (state: Bihar, income: below 2LPA) — verify scheme recommendations
Regional language test: Switch to Tamil — verify all responses are in Tamil
Mobile responsive test: Open on smartphone — verify voice button, upload, and chat all work
Cold start test: Wait 20 minutes, then query — verify loading state shows while Render wakes up
